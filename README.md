### csv-display-tool

## Installation

	```
    git pull && composer install
  ```
	## Use The Script
	
	#If No Symfony
	```
    php CsvTool.php csv:display-csv Command/products.csv --json(if you want json data)
  ```
	
	## If In Symfony
	
	```
    php bin/console csv:display-csv Command/products.csv --json(if you want json data)
  ```
